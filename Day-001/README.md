# Day 001 — Arrays: Foundation

**Topic:** Arrays — declaration, traversal, basic operations
**Status:** Completed

## What I learned:
- Arrays store elements in contiguous memory
- Indexing starts at 0 in Java
- Random access is O(1), linear search is O(n)
- arr.length gives size — no parentheses!
- Two pointer technique for reversing
