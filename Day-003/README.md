# Day 003 — Sliding Window (Fixed Size) and Two Pointers

**Topic:** Fixed-size sliding window, two pointer technique intro
**Status:** ✅ Completed

## What I learned today
- Fixed window sliding turns O(n*k) into O(n) by reusing the
  previous window's computation
- Two pointers avoid nested loops on sorted data
- Sliding window is a special case of two pointers
