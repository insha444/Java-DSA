# Day 002 — 2D Arrays and Prefix Sums

**Topic:** 2D array traversal, transpose, prefix sum technique
**Status:** Completed

## What I learned today
- 2D arrays are arrays of arrays — matrix.length gives rows
- Transpose swaps matrix[i][j] with matrix[j][i]
- Prefix sum turns repeated range-sum queries from O(n) into O(1)
- This is a huge pattern for sliding window and subarray problems later
